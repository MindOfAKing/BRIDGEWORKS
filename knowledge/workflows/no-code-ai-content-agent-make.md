## What it is
A tutorial for building an AI agent on Make.com that automates personal branding content creation. No coding required. The agent handles content ideation, drafting, and posting steps via connected modules.

## How it works
1. Open Make.com and start a new scenario
2. Add a trigger (schedule, RSS feed, or manual)
3. Connect an AI module (GPT or Claude) with a content prompt
4. Route output to publishing destinations (LinkedIn, Twitter, email)
5. Add a review step if needed before auto-post
