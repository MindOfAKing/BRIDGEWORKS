## What it is
Codex is described as an all-purpose interface for AI agents. The full course covers Codex Basics, Projects setup, and usage patterns for non-technical users. It sits above individual AI tools as a unified control layer.

## How it works
1. Install Codex and configure Projects
2. Connect your AI agents and workflows to Codex as a central hub
3. Issue instructions via chat or command interface
4. Monitor agent outputs from one dashboard
